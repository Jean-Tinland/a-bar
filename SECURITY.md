# Security Policy

## Supported Versions

Only `a-bar` latest version is supported (latest release available on GitHub).

## Reporting a Vulnerability

If you think you have detected a vulnerability, please open a new issue in `a-bar`'s repo, [here](https://github.com/Jean-Tinland/a-bar/issues).

Alternatively, you can contact me directly via the contact form on my website: [Jean Tinland - Contact](https://www.jeantinland.com/contact/).
